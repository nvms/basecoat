Retrieving data from dropdowns

When a dropdown value is changed by a click, the input element embedded inside the dropdown has its value updated to reflect the data-value attribute of the selected item. You can then proceed to pull that value from the input by referencing it by id or name as you traditionally would. This works much like pulling data from an input field, because that's exactly what you're doing.
