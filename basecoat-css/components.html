Buttons

The "small" versions of each button are actually greater in width than the medium buttons, but have a shorter height. I simply prefer them this way.

Below, the size classes in order of appearance for each set of buttons is as follows: .btn-lg, .btn, .btn-md and .btn-sm.

Form components

All of the form components are designed with readability in mind. When you look at an input field, you should immediately know that it's a field which requests input.

Placeholder & label

 

<!-- With placeholder -->
<div class="form-input-wrapper">
     <input type="text" id="user" class="form-input " placeholder="Username" required>
</div>
 
<!-- With label -->
<div class="form-input-wrapper">
     <input type="text" id="user" class="form-input" required>
     <label for="user">Username</label>
</div>
 
<!-- With placeholder and label -->
<div class="form-input-wrapper">
