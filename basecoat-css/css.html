Grid

For demonstration purposes, a three-pixel gutter in the form of a white border has been added around each grid column below. Normally, this gutter does not exist and each grid column will use the maximum amount of space available.

You can arrange blocks in any way you would like. Each row will fold automatically after the maximum number of blocks is reached.

A sixteen-column and twelve-column grid are available.

Lists

The style for all types of standard list elements have been largely untouched.

Ordered

ol > li

  1. List Item 1
  2. List Item 2
  3. List Item 3

Unordered

ul > li

  • List Item 1
  • List Item 2
  • List Item 3

Definition

dl > dt > dd

Definition term title 1
Definition term description
Definition term title 2
Definition term description
<!-- Ordered list -->
<ol>
    <li>List Item 1</li>
    <li>List Item 2</li>
    <li>List Item 3</li>
</ol>
 
<!-- Unordered list -->
<ul>
    <li>List Item 1</li>
    <li>List Item 2</li>
    <li>List Item 3</li>
</ul>

</dl>
